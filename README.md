# 🏡 eRTKas

**eRTKas** adalah aplikasi administrasi dan pengelolaan keuangan RT berbasis **Google Apps Script** dan **Google Sheets** yang dirancang untuk membantu pengurus RT mengelola data warga, iuran, arisan, jimpitan, kas sosial, serta laporan keuangan secara mudah, transparan, dan efisien.

## 🌟 Fitur Utama

### 👥 Manajemen Warga

* Data warga
* Status aktif/nonaktif
* Pencarian data warga

### 💰 Pengelolaan Keuangan

* Iuran IPAL
* Arisan
* Jimpitan
* Kas Sosial
* Pemasukan lainnya
* Pengeluaran kas

### 📊 Laporan

* Laporan pemasukan
* Laporan pengeluaran
* Rekap saldo kas
* Riwayat pembayaran warga
* Laporan bulanan
* Export PDF
* Export Excel
* Share melalui WhatsApp

### 📱 Akses Multi Platform

* Desktop
* Smartphone
* Tablet

## ✨ Keunggulan

* Interface sederhana dan mudah digunakan.
* Data tersimpan aman di Google Sheets.
* Tidak memerlukan database tambahan.
* Dapat digunakan kapan saja selama terhubung ke internet.
* Transparansi keuangan RT lebih terjaga.
* Laporan otomatis dan mudah dibagikan.

## 📂 Struktur Data

Aplikasi menggunakan beberapa sheet utama:

| Sheet            | Fungsi                |
| ---------------- | --------------------- |
| Master_Warga     | Data seluruh warga    |
| Master_Pertemuan | Data pertemuan/arisan |
| Pembayaran       | Data pembayaran warga |
| Pengeluaran      | Data pengeluaran kas  |
| Setting          | Konfigurasi aplikasi  |

---

## 🚀 Cara Menggunakan

1. Buka aplikasi melalui Web App.
2. Login menggunakan akun yang telah diberikan akses.
3. Kelola data warga.
4. Catat pembayaran sesuai jenis iuran.
5. Tambahkan pengeluaran jika diperlukan.
6. Lihat laporan secara otomatis.
7. Bagikan laporan dalam format PDF, Excel, atau WhatsApp.

## 🎯 Tujuan Aplikasi

eRTKas dikembangkan untuk membantu pengurus RT dalam:

* Digitalisasi administrasi keuangan RT.
* Mengurangi pencatatan manual.
* Mempermudah monitoring pembayaran warga.
* Menyediakan laporan keuangan yang transparan.
* Meningkatkan efisiensi administrasi lingkungan.

---

## 🔒 Keamanan

* Data tersimpan di Google Sheets milik administrator.
* Hak akses dapat diatur melalui Google Apps Script.
* Riwayat transaksi tersimpan sebagai arsip.
